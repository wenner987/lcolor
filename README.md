# lcolor
lcolor is a command line tools, it will support output log with color.

## How to build lcolor
Build lcolor with bazel:
```sh
bazel build //:lcolor
```

Build lcolor with cmake:
```sh
mkdir bin && cd bin

cmake ../ && make -j16
```
